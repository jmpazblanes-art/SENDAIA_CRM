
# Manual de Usuario - SendaIA CRM

## Acceso al Sistema
Acceder a `crm.sendaia.es` (o url de desarrollo) e iniciar sesión con credenciales corporativas.

## Módulos Principales

### 1. Clientes (Pipeline)
- **Vista Kanban**: Arrastra clientes entre columnas (Lead, Qualified, Active, Closed) para cambiar su estado.
- **Ficha de Cliente**: Haz clic en el nombre para ver detalles, historial y documentos.
- **Llamada IA**: Usa el botón "Llamar Agente IA" en la tarjeta para iniciar una llamada de cualificación automática.

### 2. Citas (Calendario)
- Visualiza tus próximas reuniones.
- Usa "Nueva Cita" para agendar manualmente o envía el link de Cal.com al cliente.

### 3. Facturación
- Sube facturas en PDF para extracción automática de datos.
- Revisa el estado de cobro (Pagado, Pendiente, Vencido).

### 4. Equipo
- Directorio de empleados con roles y especialidades.
- Contacto rápido via email o teléfono.

### 5. Chatbot Asistente
- Usa el botón flotante (esquina inferior derecha) para consultar dudas operativas o buscar datos rápidamente.
- Ejemplo: "¿Quién es el cliente con deuda pendiente?"

## Soporte
Contactar con Pachi (Founder) o equipo técnico para incidencias.
